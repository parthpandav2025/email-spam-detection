# 📧 Email Spam Detection

A Machine Learning based Email Spam Detection System built using Python and FastAPI.

This project classifies emails/messages as **Spam** or **Not Spam (Ham)** using Natural Language Processing (NLP) and Machine Learning techniques.

---

## 🚀 Features

- Detects spam emails/messages
- NLP text preprocessing
- Machine Learning model integration
- FastAPI backend
- Easy API testing

---

## 🛠️ Tech Stack

 - pandas 
 - numpy
 - matplotlib
 - seaborn
 - scikit-learn
 - nltk
 - fastapi
 - pydantic
 - uvicorn
